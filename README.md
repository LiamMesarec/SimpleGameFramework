# SimpleGameFramework
 
Needs SDL, SDL Image, SDL TTF, SDL mixer to work.

Compiled with: 
    -Wall, -Wextra, -pedantic, -std=c++20
    -lmingw32, -lSDL2main, -lSDL2, -lSDL2_image, -lSDL2_ttf, -lSDL2_mixer
